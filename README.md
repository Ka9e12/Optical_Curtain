# 🎭 Optical Curtain（透明物理幕布）

> 一扇悬在空中的幕布。捏住它，撕开它。

Optical Curtain 是一个运行在浏览器端的交互装置。它在你的屏幕中央垂下一张由数百个粒子编织而成的物理幕布，摄像头实时追踪你的手势——捏合手指抓住幕布，向上猛甩将它揉皱堆积，向下拉扯让它重新绷紧展开。幕布背后是实时摄像头画面，褶皱处会透出背后的景象。

---

## ✨ 核心亮点

### 🖐️ 手势抓取幕布

基于 Google MediaPipe 手势追踪，通过拇指与食指的距离判断「捏合」动作。捏住幕布任意位置即可拖拽拉扯，松手瞬间幕布根据甩动速度产生不同的物理反馈——向上甩则堆积揉皱，向下甩则拉伸回弹。

### 🪡 粒子级物理模拟

幕布由 38×35 共 1330 个粒子通过杆约束连接而成，基于 Verlet 积分进行实时物理解算。每一帧粒子都受重力、摩擦力、风力噪声和弹性恢复力的共同作用，幕布的每一个褶皱都真实计算而非预制动画。

### 🎥 录制交互过程

内置了一个简易的 Canvas 录制模块，点击录制按钮可将当前的操作过程保存为视频文件。录制完成后会弹出预览窗口供回放确认。

> ⚠️ 内置录制模块目前还比较简陋，推荐优先使用设备自带的录制方案：macOS/iOS 可使用系统录屏（快捷键或控制中心），Windows 可使用 Xbox Game Bar（Win+G），Android 和 iOS 均内置了屏幕录制功能。第三方工具如 OBS、Kap 也是很好的选择。

### 📁 自定义背景

除了实时摄像头画面，还支持上传本地图片或视频作为幕布背后的内容。上传后幕布会自动适配背景的长宽比。

### 🎛️ 其他控制

- **🔄 翻转**：镜像翻转摄像头画面
- **🔍 缩放**：拖动滑块调节幕布尺寸
- **🙈 净界模式**：隐藏所有 UI，沉浸观赏
- **🖱️ 鼠标回退**：没有摄像头时可用鼠标拖拽幕布

## 🛠️ 技术内幕

- **Verlet 粒子弹簧系统**：幕布顶部一排粒子固定作为挂点，其余粒子通过迭代约束求解（12 轮）保持结构稳定。捏合时以手部坐标为中心，半径 150px 内的粒子被直接拖拽；松手时根据手部垂直速度触发「向上堆积」或「向下回弹」两种状态。

- **手势防抖与平滑**：手指坐标经过指数平滑（smoothing factor 0.4），单帧最大跳跃限制 150px，防止摄像头噪点导致幕布剧烈抖动。左右手通过最短距离匹配保持身份连续性，避免交叉时跳变。

- **多平台兼容**：针对 iOS Safari 的若干特殊处理——Canvas 录制时主动拉取帧数据，摄像头元素使用 1px 占位而非 display:none 以确保 iOS 持续解码视频帧。录制编码按 H.264 MP4 到 WebM 的优先级自动选择。

## 🚀 快速开始

纯前端实现，无需搭建任何开发环境。

```bash
git clone https://github.com/Ka9e12/Optical_Curtain.git
cd Optical_Curtain
```

双击打开 `index.html` 即可在浏览器中运行。

> 💡 请允许浏览器获取摄像头权限，否则只能使用鼠标拖拽幕布。

你也可以直接在线体验：**[ka9e12.github.io/Optical_Curtain](https://ka9e12.github.io/Optical_Curtain/)**
